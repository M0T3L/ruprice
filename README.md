## Requirements:
- alacritty
- bspwm
- sxhkd
- htop
- neofetch
- neovim
- picom
- polybar
- rofi
- nitrogen
- zathura

## Preview
![screenshots](preview.jpg)

## To install:
1. install requirements above
2. create `.config` file on your home/*yourUSER* folder
>`mkdir .config`
3. clone this repository in `.config`
>`git clone https:/github.com/moteloner/ruprice`
4. edit your `.xinitrc`
5. and `startx`
